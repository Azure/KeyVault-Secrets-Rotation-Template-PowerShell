# <Service Type> keys rotation ARM templates

This template creates below components to help demonstrate <Service Type> keys rotation in Key Vault using Function and Event Grid notification.

### <Service Type> keys rotation functions ARM template:

Components:

- App Service Plan
- Function App with access to Key Vault and <Service Type> 
- Functions to rotate <Service Type> keys
- Event Subscription
